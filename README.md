## Installation
1. Clone this repository

```sh
git clone https://github.com/dimassagngsptr/Peworld.git
```

2. Install all of the required modules
using NPM
```sh
npm install
```
using Yarn
```sh
yarn add
```

3. Run this command to run the project
using NPM
```sh
npm run dev
```
using Yarn
```sh
yarn dev
```
